Назначение программы:

Назначением данной программы является приобритение орехов и семян.

Условия применения:

  1. Наличие сервера для работы данного сайта. В нашем случае это OpenServer.
  2. Наличие базы данных в приложении phpMyAdmin.
  3. Наличие ссылки на удалённую библиотеку jQuery.

Описание задачи:

    Посредством языков разметки и стилей
  • HTML
  • CSS
    Посредством языков программирования и технологии
  • php
  • js + jQuery
  • AJAX
необходимо создать интернет магазин.

Входные и выходные данные:

Входные данные беруться из базы данных, которые были занесены раньше.

Выходные данные - это изменённое содержимое базы данных.
